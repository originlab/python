Description

Set LabTalk string variable

Syntax

LT_set_str(var, value)

Parameters

var
variable name
value
string variable value

Return

True if success, and False if it fails.

Examples

EX1

import PyOrigin
PyOrigin.LT_set_str('strvar','example')
PyOrigin.LT_get_str('strvar')
