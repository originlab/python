Description

Get string value of the TreeNode

Syntax

GetStrValue( Index = "")

Parameters

Index

Return

string value of the TreeNode

Examples

import PyOrigin
NewTree = PyOrigin.EmptyTree()
if(NewTree.IsValid()):
    NewTree.SetName('NEW')
    NewTree.SetStrValue('TEST')
    print("%s" %(NewTree.GetStrValue()))
